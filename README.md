# sharememo-privacy# プライバシーポリシー（Privacy Policy）

本アプリ「シェアモ」（以下、「本アプリ」）は、ユーザーのプライバシーを尊重し、以下の方針に基づいて情報を取り扱います。

## 1. 収集する情報
本アプリでは、以下の情報を取り扱う場合があります。

- アプリ内で作成されたメモデータ
- アプリの動作改善のための匿名の利用状況情報

※ 個人を特定できる情報（氏名、住所、メールアドレス等）は収集しません。

## 2. 情報の利用目的
取得した情報は、以下の目的で使用します。

- アプリ機能の提供および改善
- 不具合修正および品質向上

## 3. 情報の第三者提供
法令に基づく場合を除き、第三者に情報を提供することはありません。

## 4. 広告・課金について
本アプリにはアプリ内課金機能があります。課金処理は Apple Inc. が提供する仕組みを通じて行われます。

## 5. プライバシーポリシーの変更
本ポリシーは必要に応じて変更されることがあります。
